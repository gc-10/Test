# Tenuta Tigliano

Sito vetrina statico per **Tenuta Tigliano**, azienda agricola dedicata alla vendita e degustazione di vini a Peccioli (Pisa).

## Contenuto del sito

- Home page con presentazione della tenuta
- Pagina "Cosa facciamo" con descrizione di vini, degustazioni e prodotti
- Pagina "Dove trovarci" con indirizzo, orari e mappa
- Pagina "Contattaci" con form e link ai canali social

## Tecnologie usate

- HTML
- CSS
- JavaScript

## Struttura del progetto

```text
.
|-- index.html
|-- css/
|   `-- style.css
|-- js/
|   |-- script.js
|   `-- notifica.js
|-- pages/
|   |-- cosa-facciamo.html
|   |-- contattaci.html
|   `-- dove-trovarci.html
`-- img/
```

## Come aprirlo in locale

Non serve nessuna installazione.

1. Apri `index.html` nel browser.
2. In alternativa, usa un server locale semplice come Live Server.

## Note

- Il menu di navigazione e responsive e viene gestito da `js/script.js`.
- Il form contatti usa **FormSubmit**, quindi non c'e un backend nel progetto.
